# Anime color scale (discrete)

Anime color scale (discrete)

## Usage

``` r
scale_color_anime(palette = "Asoul", alpha = 1, reverse = FALSE, ...)
```

## Arguments

- palette:

  Palette name.

- alpha:

  Alpha transparency.

- reverse:

  Reverse palette order.

- ...:

  Passed to
  [`ggplot2::discrete_scale()`](https://ggplot2.tidyverse.org/reference/discrete_scale.html).
