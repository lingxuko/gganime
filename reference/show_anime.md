# Show a palette

Title: "Palette of " Each color block shows: "#RRGGBB" centered in the
block

## Usage

``` r
show_anime(palette = "Asoul", n = NULL, reverse = FALSE, alpha = 1)
```

## Arguments

- palette:

  Palette name.

- n:

  Number of colors to show. Defaults to the palette max.

- reverse:

  Reverse palette order.

- alpha:

  Alpha transparency
